# בוט ויקיטקסט - שולחן ערוך
בוט שנועד להוסיף הפניות למפרשים במהדורת שולחן ערוך של ויקיטקסט.

## תלויות
הקוד משתמש ב[https://www.mediawiki.org/wiki/Manual:Pywikibot]pywikibot לצורך התקשורות עם ויקיטקסט, וב[https://wikitextparser.readthedocs.io/en/latest/README.html#templates]wikitextparser לצורך ניתוח הטקסט המתקבל מויקיטקסט.
הדרך הפשוטה ביותר להתקין את שניהם היא pip, הוראות נוספות נמצאות בקישורים למעלה.
