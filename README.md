# בוט ויקיטקסט - שולחן ערוך
בוט שנועד להוסיף הפניות למפרשים במהדורת שולחן ערוך של ויקיטקסט.

## איך אפשר לעזור
כרגע הקוד של הבוט עצמו עובד בגדול, ומה שנותר הוא להוסיף בקובץ conversion_data.py את קיצורי המפרשים וביטויים רגולריים המתאימים לצורה בה מופיעים הדיבורים המתחילים בתבנית \{\{משע\}\} בדפי המפרשים בויקיטקסט. יש התחלה בקובץ, וכעת צריך להרחיב אותו.

## תלויות
הקוד משתמש ב[pywikibot](https://www.mediawiki.org/wiki/Manual:Pywikibot) לצורך התקשורות עם ויקיטקסט, וב[wikitextparser](https://wikitextparser.readthedocs.io/en/latest/README.html#templates) לצורך ניתוח הטקסט המתקבל מויקיטקסט.
הדרך הפשוטה ביותר להתקין את שניהם היא pip, הוראות נוספות נמצאות בקישורים למעלה.
